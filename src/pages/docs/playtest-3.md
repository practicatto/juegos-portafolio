Este playtest corresponde a la versión 0.3 del juego Encuentra al títere.

## Juego inicial

- El juego comienza con 4 cartas por cada jugador, estas cartas son escogidas de manera aleatoria por lo que tendrían cartas de acción o leyes.
- La carta que roban de la baraja la deben jugar obligatoriamente.
- Las condiciones de victoria: 3 leyes aprobadas para el gobierno; 7 leyes aprobadas para la oposición.
- El presidente tiene voto doble durante una votación y puede observar las 3 primeras cartas de la baraja para decidir cuál carta robar.

## Cambios desde la versión 0.3

- Las condiciones de victoria cambian a: 3 leyes y la presidencia para el gobierno; 6 leyes para la oposición.
- El moderador es eliminado del juego.
- Para ganar siendo oposición, se realiza una votación para revelar a todos los jugadores. Si se
  ha cumplido la condición de victoria de la oposición, ellos ganan. En caso contrario, gana el gobierno.
- Se crean las cartas de juego rápido para usar cartas de acción durante el turno de otros jugadores.
- Cuando un jugador es eliminado, las leyes aprobadas aún cuentan en el juego.
